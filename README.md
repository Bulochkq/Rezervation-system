# Сторінка бронювання велосервісу Velocity (Bratislava)

Цей проєкт — сторінка для онлайн-бронювання ремонту та обслуговування велосипедів і електровелосипедів у мережі Velocity Servis / PRO CYCLING у Братиславі. 

---

## 📍 Наші філії (Локації)

На сторінці інтегровано дві основні точки обслуговування:
1. **Velocity Servis Petržalka**
   * Адреса: Medveďovej 1/A, Bratislava
   * Посилання на карту: [Google Maps](https://maps.app.goo.gl/Czi3X4quc3b4jX8B7)
   * Система бронювання: `servis.procycling.sk`

2. **Velocity Servis WESTEND**
   * Адреса: Lamačská cesta 3B, Westend plazza, Bratislava (переїхала з Lamačská cesta 8/A)
   * Посилання на карту: [Google Maps](https://maps.app.goo.gl/goiYpc4C3Gp2JPTu7)
   * Система бронювання: `servis.velocity.sk`

## 🛠️ Особливості реалізації

Я налаштував цю сторінку так, щоб вона була максимально швидкою, зручною для клієнтів та добре індексувалася пошуковцями:

* **Локальне SEO (Schema.org)**: Вся детальна інформація про наші точки (адреси, точні GPS-координати, контакти та години роботи) прописана у форматі JSON-LD. Це допомагає Google правильно зчитувати дані та відображати філії у локальному пошуку й на картах.
* **Оптимізація завантаження**: Календарі бронювання підвантажуються через iframe динамічно — тільки після натискання кнопки «REZERVOVAŤ TERMÍN». Завдяки цьому сторінка відкривається миттєво, не чекаючи завантаження важких сторонніх скриптів календаря.
* **Зручність на мобільних телефонах**: Інтерфейс модального вікна адаптовано під екрани смартфонів. Календар розтягується на всю ширину дисплея, а елементи керування розташовані так, щоб їх не перекривала плашка адресної стрічки браузера на iOS чи Android.
* **SEO для зображень**: Усі картинки мають описові атрибути `alt`. Фонове зображення в шапці та картинки філій також оптимізовані для пошукових роботів.
